# lofiland-hyprland
My hyprland config


Requirements:
- rofi ()
- kitty
- catnap (https://aur.archlinux.org/catnap-git.git)
- waybar
- btop
- swaybg


Font:
- Starborn
- JetbrainsMono

run:
```
sudo pacman -S kitty rofi waybar btop discord zsh swaybg
git clone https://github.com/sophin-2010/lofiland-hyprland.git ~/.config
mv ~/.config/.zshrc ~/
chsh -s /usr/bin/zsh
```
